# 知之为知之，不知问 AI｜团队材料站

这是“飞书 AI 未来人才大赛”的团队材料介绍网站，用于集中展示团队概要、人生实验室方案、成员介绍、调研证据、竞品研究及相关补充材料。

## 核心理念

**人生实验室**：帮助学生把模糊兴趣变成可验证假设，通过一次低成本真实体验获得新证据，再修正下一步判断。

## 本地运行

项目统一使用 Node.js 24。使用 nvm 时，可在项目目录运行 `nvm use` 切换版本。

```bash
npm install
npm run dev
```

访问 `http://localhost:3000` 即可查看网站。

## 常用命令

```bash
npm run dev      # 启动开发环境
npm run build    # 构建生产版本
npm run start    # 启动生产版本
```

## 目录说明

- `app/`：页面入口与全局样式
- `components/`：页面区块和通用组件
- `download/`：可下载的团队原始材料
- `public/`：静态资源及构建时同步后的下载文件
- `scripts/sync-downloads.mjs`：将 `download/` 内容同步到公开下载目录

运行开发、构建或生产启动命令前，项目会自动同步下载材料。
